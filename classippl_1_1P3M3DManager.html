Detailed Description

template<typename T, unsigned Dim, class fc>
class ippl::P3M3DManager< T, Dim, fc >

A template class for managing Particle-Particle Particle-Mesh (P3M/PPPM) simulations.

The P3M3DManager class is a template class that extends the functionality of the BaseManager class for P3M simulations. It provides functionality for particle-particle, particle-to-grid and grid-to-particle operations

Template Parameters
Tthe data type for simulation variables
DimThe dimensionality of the simulation (here 3D only)
fcThe field container type
